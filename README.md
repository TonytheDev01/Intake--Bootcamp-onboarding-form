# Intake--Bootcamp-onboarding-form
INTAKE is a clean, semantic HTML onboarding form designed to simulate how real-world bootcamps collect student information during enrollment.  This project focuses on structure, accessibility, and UX-first form design, without relying on CSS frameworks or JavaScript.

GOAL OF THE PROJECT
The goal of this project is to strengthen my understanding of:
Semantic HTML structure
Accessible form design
Proper input labeling and grouping
Realistic onboarding flows used in products and bootcamps.

TECH STACK
HTML5 only

FEATURES
Clear, structured onboarding flow
Semantic layout using header, main, section, and footer
Accessible form inputs with proper labels
Logical grouping using fieldset and legend
Realistic bootcamp-related questions
Mobile-friendly structure (CSS to be added later)
